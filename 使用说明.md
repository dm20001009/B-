# B站视频逐字稿生成工具 — 使用说明

把 B 站视频链接粘贴进网页，自动生成带标点、分好段的中文逐字稿。
支持复制全文和下载 txt 文件。

---

## 一、电脑要求

- Windows 10 / 11
- 硬盘剩余空间 5GB 以上（语音识别模型较大）
- 能正常上网

## 二、首次安装（只需做一次，约 10~20 分钟）

### 第 1 步：安装 Python

1. 打开 https://www.python.org/downloads/ 下载最新版 Python
2. 运行安装程序时，**务必勾选底部的 "Add python.exe to PATH"**（最关键的一步！）
3. 点 Install Now 完成安装

### 第 2 步：首次启动

1. 把整个 `bili-transcript-app` 文件夹复制到电脑任意位置（建议放在 C 盘用户目录或桌面）
2. **双击文件夹里的 `启动.bat`**
3. 第一次运行它会自动做三件事，请耐心等待：
   - 自动安装 ffmpeg（音频处理工具）——装完后按提示**重新双击一次** `启动.bat`
   - 自动安装 Python 依赖包（几分钟）
   - 启动服务并自动打开浏览器

> 看到浏览器打开 `http://localhost:5000` 页面，就说明安装成功了。

### 第 3 步：配置 B 站登录凭证（cookies.txt）

B 站要求登录后才能下载视频，需要导出一次你的登录凭证：

1. 在 Chrome 或 Edge 浏览器安装扩展：**Get cookies.txt LOCALLY**
   （商店地址：https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc）
2. 打开 https://www.bilibili.com 并**登录你自己的账号**（确认右上角有头像）
3. 停留在 B 站页面上，点击浏览器右上角的扩展图标 → 点 **Export**
4. 把导出的文件重命名为 `cookies.txt`，放进 `bili-transcript-app` 文件夹
5. 刷新工具网页，右上角显示"Cookie 已就绪"即可

> 凭证有效期约 30 天，平时正常刷 B 站会自动续期。如果某天工具提示 412 错误，重新导出一次即可。

---

## 三、日常使用

1. **双击 `启动.vbs`**（无黑窗口，3 秒后自动打开浏览器）
2. 粘贴 B 站视频链接（长链接、短链接、稍后再看链接都可以）
3. 点"生成逐字稿"，等待完成
4. 完成后可以**复制全文**或**下载 .txt**

### 两种处理路径（自动选择）

| 路径 | 触发条件 | 耗时 |
|------|----------|------|
| 提取 B 站内置字幕 | 视频本身带字幕/AI字幕 | 几秒钟 |
| Whisper 语音识别 | 视频没有字幕 | 10分钟视频约 3~10 分钟 |

### 识别精度选择（仅语音识别时生效）

- **推荐（medium）**：日常使用，准确率高
- **平衡（small）**：速度快一倍，准确率略降
- **最高精度（large）**：最准但很慢，重要内容才用

> 第一次用语音识别时会自动下载模型（medium 约 1.5GB），只下载一次。
> 第一次生成逐字稿时还会自动下载标点模型（约 400MB），也只下载一次。

## 四、常见问题

**Q：页面打不开 / 显示无法访问**
双击 `启动.vbs` 后等 5 秒再刷新浏览器。还不行就双击 `启动.bat` 看窗口里的报错信息。

**Q：提示 412 错误 / B 站拒绝访问**
cookies.txt 过期了。按"第 3 步"重新导出一次替换即可（注意导出前要登录 B 站）。

**Q：转出来没有标点？**
正常情况会自动补标点。如果没有，可能是首次下载标点模型失败，检查网络后重试一次。

**Q：转录很慢**
语音识别靠 CPU 计算，视频越长越慢。可以把识别精度调低一档，或者优先选带字幕的视频（秒出结果）。

**Q：大会员专属视频能用吗？**
你的账号有大会员就可以（凭证里包含会员身份）。

## 五、文件说明

```
bili-transcript-app\
├── 启动.vbs        日常启动（推荐，无黑窗口）
├── 启动.bat        首次安装 / 排查问题用
├── app.py          主程序
├── punctuate.py    中文标点恢复模块
├── templates\      网页界面
├── requirements.txt 依赖清单
└── cookies.txt     你的 B 站登录凭证（自己导出，见第 3 步）
```

---

## ⚠ 重要提醒

- **cookies.txt 等同于你的 B 站账号钥匙，绝对不要发给任何人！**
  把工具分享给朋友时，**删掉文件夹里的 `cookies.txt` 再发**，让对方按第 3 步导出他自己的。
- 本工具仅供个人学习使用，请勿用生成的逐字稿侵犯视频作者版权。
